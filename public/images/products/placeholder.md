## Adding Images

1. Add your product images to this directory
2. Use descriptive filenames (e.g., `rose-lip-balm.jpg`)
3. Recommended image size: 800x800px
4. Supported formats: JPG, PNG, WebP

## Using Placeholder Images

If you don't have images yet, you can use placeholder services:
- https://placehold.co/800x800
- https://picsum.photos/800/800
- https://via.placeholder.com/800

Or use the default fallback in the code: `/images/default-product.jpg`
